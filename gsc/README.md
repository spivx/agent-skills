# GSC — Google Search Console

Connects directly to the Google Search Console API to fetch live search performance data and deliver actionable SEO insights.

**Live API data** — real metrics, not manual data pasting. **Zero dependencies** — pure Node.js. **Built-in analysis** — CTR benchmarking, opportunity detection, prioritized recommendations.

## Install

```bash
npx skills add spivx/agent-skills --skill gsc
```

## What You Get

- Executive summary with health assessment against CTR benchmarks
- Top queries and pages with performance categorization
- High-impression/low-CTR keywords ripe for optimization
- Page-2 rankings one push away from page 1
- Actionable recommendations prioritized by effort vs impact
- Period-over-period trend analysis on request
- RTL-aware — Hebrew/Arabic sites get a formatted HTML report that auto-opens in your browser
